# Tubesandat - Data Analysis Project

## Overview

This repository contains a comprehensive data analysis project developed by **Group 9 Class K02**. The project focuses on analyzing datasets using R and applying statistical methods to derive meaningful insights.

## Project Description

**Tugas Besar Analisis Data Kelompok 9 K02** is a major assignment for MA2181 Data Analysis course. This project demonstrates data exploration, visualization, statistical testing, and interpretation techniques using R.

## Project Objectives

- Conduct exploratory data analysis (EDA) on provided datasets
- Apply appropriate statistical methods and hypothesis testing
- Create visualizations to communicate findings
- Document analysis process and conclusions

## Technology Stack

- **Language**: R
- **Version Control**: Git & GitHub

## Project Structure

```
tubesandat/

├── data/                  # Raw and processed datasets
├── scripts/               # R analysis scripts
├── output/                # Generated plots and results
└── reports/               # Analysis reports and findings
```

## Getting Started

### Prerequisites

- R (latest version recommended)
- Required R packages (see `scripts/setup.R`)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/realfeeds/tubesandat.git
   cd tubesandat
   ```

2. Install required packages:
   ```R
   source("scripts/setup.R")
   ```

## Usage

Run the analysis scripts:
```R
# Load and explore the data
source("scripts/01_data_loading.R")

# Perform analysis
source("scripts/02_analysis.R")

# Generate visualizations
source("scripts/03_visualization.R")
```

## Key Analyses

- Data cleaning and preprocessing
- Descriptive statistics
- Statistical hypothesis testing
- Data visualization and interpretation

Analysis results, plots, and findings are saved in the `output/` directory.
